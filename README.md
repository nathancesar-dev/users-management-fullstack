# 👥 Gerenciador de Usuários - Full Stack

Este é um sistema completo de gerenciamento de usuários desenvolvido durante a formação **DevClub**. A aplicação conta com uma interface moderna em React e um backend robusto em Node.js com persistência no MongoDB via Prisma.

---

## 🚀 Tecnologias

Este projeto foi construído utilizando as seguintes tecnologias:

### **Frontend**
*   **React.js** (Vite)
*   **Styled Components** (Estilização)
*   **Axios** (Consumo de API)
*   **Yarn** (Gerenciador de Pacotes)

### **Backend**
*   **Node.js** & **Express**
*   **Prisma ORM** (Modelagem de dados)
*   **MongoDB Atlas** (Banco de dados na Nuvem)
*   **CORS** (Segurança e integração)
*   **NPM** (Gerenciador de Pacotes)

---

## 📦 Estrutura do Projeto

O repositório está organizado no modelo **Monorepo**:

- `/client`: Código-fonte da interface (Frontend).
- `/server`: Código-fonte da API e configurações do banco (Backend).
